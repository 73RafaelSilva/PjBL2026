Projeto referênte a formatica de Programação Orientada a Objetos, realizada pelos alunos da PUCPR do professor Pedro Horchulhack .

O Projeto se trata de um gerenciador de Comunidades, onde pessoas podem criar ou entrar em grupos, onde haveram eventos sendo criados que elas podem dizer se vão ou não.

Uma pessoa pode criar n grupos (o criador ganha titulo de adm);
n pessoas podem ingressar no grupo atravez de convite;
Em um grupo, o adm pode criar Eventos;
O adm do grupo pode promover n pessoas a adm;
As pessoas do grupo podem marcar presença nos evenos;
o evento gera lista de presença para controle;
